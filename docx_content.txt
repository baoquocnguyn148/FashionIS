1. TỔNG QUAN DỰ ÁN
1.1 Mô tả hệ thống
StyleVibe Fashion MIS là hệ thống thông tin quản lý tích hợp toàn diện dành cho chuỗi bán lẻ thời trang địa phương. Hệ thống được xây dựng theo mô hình 3 tầng TPS → MIS → DSS, cung cấp đầy đủ nghiệp vụ từ bán hàng tại quầy, quản lý tồn kho, báo cáo kinh doanh đến hỗ trợ ra quyết định tự động dựa trên ML.

4. THIẾT KẾ CƠ SỞ DỮ LIỆU
4.1 Tổng quan &amp; Nguyên tắc thiết kế (BA Perspective)
Nguyên tắc
Chi tiết &amp; Lý do
Chuẩn hóa 3NF
Tất cả bảng đạt chuẩn 3NF. Tách ProductSku riêng khỏi Product để tránh repeating groups theo size/color. Tách OrderDetail khỏi Order để tránh phụ thuộc bắc cầu.
Soft Delete
Dùng cột IsDeleted TINYINT thay vì DELETE vật lý. Đảm bảo tính toàn vẹn dữ liệu lịch sử, phục hồi khi cần, audit trail đầy đủ.
Audit Trail
BaseEntity có CreatedAt, UpdatedAt. Bảng StockAdjustment ghi toàn bộ lịch sử thay đổi tồn kho với lý do. LoyaltyTransaction ghi từng điểm tích/đổi.
Phân tách nghiệp vụ
Order (đơn bán) tách biệt hoàn toàn PurchaseOrder (đơn nhập). Inventory quản lý per-store per-SKU, không gộp chung.
Hỗ trợ MIS/DSS
SQL Views được thiết kế sẵn để layer MIS query nhanh mà không join phức tạp trong ứng dụng. Indexed foreign keys tối ưu performance.
Mở rộng tương lai
Bảng Category hỗ trợ đệ quy (ParentCategoryId). Customer có UserId FK liên kết ASP.NET Identity. Promotion table cho phép nhiều loại ưu đãi.

6.2 Các bước setup lần đầu
Bước 1 – Clone repo: git clone &lt;repo-url&gt; &amp;&amp; cd StyleVibe
Bước 2 – Mở StyleVibe.sln trong Visual Studio 2022
Bước 3 – Cập nhật ConnectionString trong src/StyleVibe.Web/appsettings.json
Bước 4 – Package Manager Console: Update-Database (chạy EF Core migrations)
Bước 5 – Chạy data seed: python dss_service/data/generate_data.py
Bước 6 – Import CSV vào SQL Server: python dss_service/data/import_to_sqlserver.py
Bước 7 – Setup Python: cd dss_service &amp;&amp; pip install -r requirements.txt
Bước 8 – Chạy FastAPI: uvicorn main:app --reload --port 8000
Bước 9 – Chạy .NET: Ctrl+F5 trong Visual Studio (port 5000)
